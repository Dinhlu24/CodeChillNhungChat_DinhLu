# Kho Lưu Trữ Luyện Tập Thuật Toán

Chào mừng bạn đến với kho lưu trữ mã nguồn cá nhân của Đình Lu! tại repo git này mình sẽ đồng hành cùng nhóm Code Chill Nhưng Chất

## Thông Tin Chung
* **Ngôn ngữ lập trình:** Java
* **Môi trường phát triển (IDE):** IntelliJ IDEA
* **Mục tiêu:** Rèn luyện kỹ năng giải quyết vấn đề, tối ưu hóa thời gian và bộ nhớ chạy code (Time & Space Complexity).

##  Cấu Trúc Dự Án
Toàn bộ mã nguồn giải thuật được gom chung vào thư mục `src`.
Để đảm bảo tính độc lập và dễ theo dõi, mỗi bài toán sẽ được đặt trong một `package` riêng biệt. Mỗi `package` bao gồm:
* File `Main.java`: Chứa mã nguồn thực thi.
* File `README.md`: Giải thích chi tiết ý tưởng thuật toán và đánh giá độ phức tạp của bài toán đó.

## Cách Chạy Thử Code
1. Clone dự án này về máy tính cá nhân.
2. Mở thư mục dự án bằng IntelliJ IDEA (hoặc bất kỳ IDE Java nào bạn quen dùng).
3. Điều hướng vào thư mục `src` và mở `package` chứa bài toán bạn muốn thử nghiệm.
4. Chạy file `Main.java` để xem kết quả in ra màn hình Console.
